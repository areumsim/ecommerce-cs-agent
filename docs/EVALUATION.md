# Evaluation
## Reasoning Validation Framework

이 문서는 이 시스템의 판단이 **논리적으로 안전한지, 일관적인지**를 검증하는 평가 체계를 설명한다.

---

## 1. 평가의 목적

이 시스템은 정확한 결과보다 **설명 가능한 판단**을 목표로 한다. 따라서 평가는 다음 질문에 답해야 한다.

- 판단이 일관적인가?
- 규칙이 자기모순을 만들지 않는가?
- 보조 모델이 판단을 침범하지 않는가?
- 설명이 판단 경로를 충분히 반영하는가?

---

## 2. 평가의 위치

평가는 추론 흐름 이후에 위치하며, 판단 결과를 변경하지 않는다.

---

## 3. 주요 평가 지표

### Precision / Recall
규칙 기반 판단의 정확성과 포괄성을 측정한다.

### Conflict Rate
동일 대상에 대해 상충되는 판단이 생성되는 비율을 측정한다.

### GNN Uplift
Rule-only 대비 GNN 보조 효과를 측정한다.

### Explainability Coverage
설명이 실제 추론 경로를 빠짐없이 포함하는지 측정한다.

---

## 4. 평가 원칙

- Read-only
- 재현 가능
- 판단 흐름 비침범
